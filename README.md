# Code Playground

Editor de código online con soporte para múltiples lenguajes, usando Judge0 para ejecución en servidor.

## Setup

1. Copia `.env.example` a `.env.local` y añade tu clave de RapidAPI para Judge0:
   ```
   JUDGE0_API_KEY=your_rapidapi_key_here
   ```

2. Instala dependencias:
   ```bash
   npm install
   ```

3. Arranca el servidor:
   ```bash
   npm run dev
   ```

## Lenguajes soportados
- PHP
- JavaScript (Node.js)
- Python
- Java
- C++
- Rust
